Quickstart for Fahed Zidani Resume REST API

Learn how to access my Resume via REST API call

Swagger: https://myresumepai.azurewebsites.net/swagger-ui/index.html#/.

Endpoints

Once you're set up with cURL or an API client, you can explore my API resume using various endpoints. Endpoints are different URLs where you can make GET requests to view the different parts of my resume.


See the available endpoints below:

  1. Resume: View Full Resumehttps://myresumepai.azurewebsites.net/api/cv/resume
  2. Projects: View information about the projects I've built.https://myresumepai.azurewebsites.net/api/cv/projects
  3. Experience: Read about my previous career experience.https://myresumepai.azurewebsites.net/api/cv/experience
  4. Contact: View my preferred contact information. https://myresumepai.azurewebsites.net/api/cv/contact
